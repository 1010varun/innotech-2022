Question: What programming language should I learn first?

Answer : You should first learn C. You need to know how to implement everything manually. I don't prefer using python in the beginning. Python is a great language without any doubt. I myself is programmer of python. But I first started with C and C++.

Question: Why we should learn to program first in C language?

Answer : It's not necessary to learn c before any other languages. You can start with Python also. But everyone recommends to start your programming journey with c language because it is most basic language and it's fundamentals are also very useful while studying other languages. So to be frank don't start with c language, just start with Python.

Question: What is best way to learn android app development and where?

Don't try to dig in too deeply in other codes at start. Your app will start to shape, with lots of bugs and not optimized code. Don't worry. At one stage you will feel a lot more confident.
